## 始于一场视觉革命

当传统的下载站还在用枯燥的列表堆砌链接时，**赛博枢纽**已经将文件仓库变成了数字艺术的展廊。这个站点不只是文件的存放地，更是一次对“下载体验”的重新定义——每一个按钮的微光、每一根网格线的流动、每一次主题切换的平滑过渡，都在诉说着：下载，也可以是一种沉浸式体验。

![](./hero-preview.jpg)

站点的核心视觉来自于**动态网格系统**：画布上的网格线持续向右下方向流动，仿佛数据流在数字海洋中缓缓漂移。鼠标悬浮时，所在网格会被高亮，形成一种“你在数据中留下坐标”的交互隐喻。背景中漂浮的粒子系统则模拟了数字尘埃，为整个空间注入呼吸感。

## 双主题：暗夜与晨曦的对话

考虑到不同用户的视觉偏好，站点内置了**暗色/亮色双主题**：

- **暗色模式**：深邃的黑色背景搭配暖橙色（`#ffa300`）作为点缀，营造出赛博空间的沉浸感，适合深夜探索。
- **亮色模式**：米白底色搭配靛蓝色（`#0052d9`），清爽明快，适合日间阅读。

主题切换不仅影响文字与卡片颜色，更会实时改变：
- 网格线的颜色
- 粒子效果的光晕
- 鼠标跟随光晕的透明度
- 滚动条的渐变配色

所有变化都通过 CSS 变量动态注入，无需刷新页面即可完成切换。

::tip
主题偏好会保存在浏览器的 `localStorage` 中，下次访问时会自动恢复你的选择。
::

## 神经档案：文章系统的赛博重构

### 文章卡片：从信息到体验

站点的博客区域（“LATEST THOUGHTS”）并非简单陈列标题和日期，而是通过以下设计让阅读变得更具仪式感：

- **渐进式入场动画**：每张卡片按顺序依次升起，带有轻微弹性，避免内容同时涌现的机械感。
- **悬浮光效**：鼠标悬停时，卡片边框泛起彩色光晕，顶部会有流光划过，模拟数据读取的视觉反馈。
- **元数据融合**：日期与分类使用等宽字体（Space Mono）和主题色，与正文形成清晰区分。

### 文章阅读：新窗口，但不断联

点击“读取更多”时，文章内容会在**新窗口中打开**，但——请注意——新窗口会**继承当前站点的主题**（暗色/亮色），并采用类似的视觉风格，包括：
- 动态网格背景（微调后的低透明度版本）
- 代码块的高亮配色与主题同步
- 标题的渐变色彩沿用主站逻辑

## 量子仓库：下载文件的新形态

下载区域（“NEURAL DOWNLOADS”）采用了与博客卡片几乎一致的视觉语言，但针对“文件”这一属性做了特别优化：

- **文件图标**：使用 Font Awesome 的图标库，每个文件可以配置专属图标（如 `fa-archive`、`fa-code` 等）。
- **元数据展示**：文件大小被突出显示，同时用 `数据库` 和 `盾牌` 图标强化“安全可靠”的心理暗示。
- **下载按钮**：采用填充渐变边框 + 波纹扩散动画，点击时有脉冲反馈，下载链接指向 `Files/` 目录下的实际文件。

当文件数量超过 3 个时，首页会显示“+N 个文件”的**展开卡片**，点击即可跳转至 `all-downloads.html` 查看全部资源。这个设计兼顾了首页的简洁性与仓库的完整性。
